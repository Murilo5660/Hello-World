## Descrição

Primeiro programa em Python, criado para exibir uma mensagem simples na tela — um passo inicial para entender o funcionamento básico da linguagem e do comando `print()`.

## Funcionalidades

1. **Exibição de mensagem** – mostra o texto “Hello World” no console.

## Comandos utilizados

* Comando básico `print()`

## Objetivo do Projeto

Aprender o funcionamento do **primeiro programa em Python**, testando o ambiente e compreendendo a estrutura inicial de um código.
